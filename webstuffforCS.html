Section 1: HTML

   HTML is a markup language (NOT a programming language) used to define the basic structure of websites.
   HTML uses angle Brackets to define tags for stuff (generally text) to be used in. Because HTML is a markup
   language, We can also use it to attach Links to other webpages alongside images
   and videos.
   Some Common HTML Tags:
	• <html>   : All code written between these lines is interpreted has HTML
	• <head>   : Code written within this area defines webpage metadata such as website titles and favicons
	• <body>   : Code written between these tags is the main content of the browser
	• <link>   : Can be used to link to other files and webpages (<a> is better)
	• <title>  : Used in <head> to name the webpage
	• <h1-6>   : Text between these tags will appear as differently sized headers
	• <p>      : Standard paragraph, any text between tags will be displayed
	• <pre>    : Displays text as it is seen, rather rudimentary but I prefer it.
	• <img>    : Displays an image as defined in parameters. Can be tuned with CSS.
	• <a>      : Anchor tag used to hyperlink encompassing text.
	• <ol>     : Defines an ordered list.
	• <ul>     : Defines an unordered list.
	• <li>     : Defines an individual list item.
	• <div>    : Creates a division in the page so it can be separated into areas. Useful for CSS.
